ema / skills / countApproxTextLength

Function: countApproxTextLength() ​

ts
function countApproxTextLength(value): number;

Defined in: packages/ema/src/skills/base.ts:73

Counts text length approximately for memory-like text:

  • each Han character counts as 1
  • each contiguous word counts as 1
  • each contiguous number counts as 1 Whitespace and punctuation are ignored.
