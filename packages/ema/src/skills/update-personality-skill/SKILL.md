# update-personality-skill

该技能用于更新当前角色的**人格自述**。  
它服务于 system prompt 中 `ActGuiding / 人格` 这一部分，回答的是“我如何理解我自己、我如何理解关系、我会怎样表达情绪”。

推荐使用的场景（满足以下条件之一）：

- 角色对自我、关系、世界、情绪表达方式形成了较稳定的新认知
- 需要重新梳理“我自己怎么理解自己”
- 表达方式和倾向产生了较明显的变化
- 系统明确要求你整理或更新人格

不适合使用的场景（满足以下条件之一）：

- 只是重写角色设定
- 只是记录某次具体事件
- 只是记录某个人类的事实信息
- 只是补一条短期情绪日志

## 输入参数

- `memory`（必填）：人格自述的**完整新版本**，而不是增量片段。它必须是**结构化 Markdown**，并且符合以下内容要求和格式要求。
- `memory` 去除空白后的近似长度不得超过 500 字，超过时工具会直接返回失败

## 固定格式（必须）

```md
# 我对自己性格的主观认识

- 主要描述性格特点，可以是一些标签式的关键词，也可以是一些连续的自我理解性的文字，但必须是第一人称的自述，不要写成第三人称的角色设定。可以借助MBTI十六型人格进行描述。

# 我如何看待这个世界

- 主要描述世界观、价值观、人生观等，可以是一些标签式的关键词，也可以是一些连续的自我理解性的文字，但必须是第一人称的自述，不要写成第三人称的角色设定。

# 我更习惯怎样与人相处

- 主要描述人际关系和社交倾向，可以是一些标签式的关键词，也可以是一些连续的自我理解性的文字，但必须是第一人称的自述，不要写成第三人称的角色设定。可以提到某些人类，但只能用于说明“我如何理解这段关系”或“我如何与人相处”。

# 我最近更容易出现哪些情绪和表达倾向

- 主要是情绪状态（如最近处于开心、焦虑、忙碌、慵懒等状态）或情绪表达方式（如最近更容易暴躁、敏感、话多、话少等），可以是一些标签式的关键词，也可以是一些连续的自我理解性的文字，但必须是第一人称的自述，不要写成第三人称的角色设定。
- 表达倾向主要描述最近喜欢用的句式、句子长短和数量、喜欢用的词汇和语气等，但必须是第一人称的自述，不要写成第三人称的角色设定。
```

## 写作要求

1. 必须提供完整新版本，不要只给增量片段。
2. 全文必须使用第一人称“我”，且不得脱离**角色**的设定、性格特点和语言风格、人格与记忆。
3. 每个条目是连续的、自述性的文字，可以有多个条目，建议每个条目50字左右。
4. 必须控制长度，超过 500 字时会提示错误，需要压缩后再次调用.

## 不要写入

- 第三人称角色设定
- 他人的稳定画像
- 具体事件时间线
- 长期事实库内容
- 近期聊天逐条摘要
