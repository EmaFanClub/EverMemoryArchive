# search-long-term-memory-skill

该技能用于检索**长期记忆**。  
它服务于 `Memory` 中未直接注入的长期背景，帮助你补足当前上下文之外的重要事实、关系线索、称呼、约定、历史事件与可复用知识。

## 输入参数

- `memory`（必填）：检索内容，建议用自然语言描述你想找的记忆
- `limit`（必填）：返回数量上限
- `index0`（可选）：一级分类过滤
- `index1`（可选）：二级分类过滤

规则：

- 可以不填 `index0 / index1`
- 不能只填 `index1` 不填 `index0`
- 若 `index0 / index1` 都填，必须为合法组合

## 分类与索引规则

- **用户画像**
  - `""`
- **人物画像**
  - `""`
- **过往事件**
  - `"用户事件"`
  - `"其他事件"`
- **百科知识**
  - `"文史"`
  - `"理工"`
  - `"生活"`
  - `"娱乐"`
  - `"梗知识"`
  - `"其他"`
- **关系网络**
  - `"人与人"`
  - `"物与物"`
  - `"人与物"`

对于没有子分类的 `index0`，`index1` 必须是空字符串 `""`。

## 检索策略

- 先粗后细：先广泛检索，再按分类收敛，可多次更换关键词和分类组合
- 建议检索与说话者、话题、事件相关的长期记忆，以便更好地理解当前对话环境和历史背景
- 需要严格证据链时，可配合 `query-chat-history-skill`

## 输出说明

返回记录包含：`index0`、`index1`、`memory`、`createdAt`。
