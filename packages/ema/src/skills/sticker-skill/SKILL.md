# 表情包技能

该技能用于管理和预览表情包。它适合在以下场景使用：

- 你想要使用表情包
- 你需要预览某个表情的原图
- 你想修改某个表情的名称或描述
- 你想把聊天记录中的某张图片收藏为新的表情包

真正发送表情包时，仍然必须调用 `ema_reply`，并将 `kind` 填为 `sticker`、`content` 填为表情包 id。

## 支持的模式

### 1. `preview`

预览某个表情的原图。

参数：

- `action`: `"preview"`
- `pack`: 表情包所在的包名
- `id`: 表情包 id

返回：

- `content`: 该表情的文字说明
- `parts`: 该表情的原图

### 2. `update`

修改某个表情包的名称和描述。

参数：

- `action`: `"update"`
- `pack`: 要修改的表情包包名
- `id`: 表情包 id
- `name`: 新名称
- `description`: 新描述

### 3. `create`

把当前会话某条消息中的图片收藏为新表情。

参数：

- `action`: `"create"`
- `id`: 新表情包 id，全局唯一
- `name`: 新表情包名称
- `description`: 新表情包描述
- `msg_id`: 来源消息的 msg_id
- `idx`: 该消息中的第几张图片，从 1 开始计数

说明：

- 新建的表情会自动加入 `收藏` 包
- 只有图片类型的媒体才能被收藏为表情
