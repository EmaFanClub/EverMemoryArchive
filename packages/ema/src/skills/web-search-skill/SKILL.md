# 联网搜索技能

此技能用于搜索外部网页信息，适合在以下场景使用：

- 你想了解某个主题的最新资料或近期动态
- 你想搜索某个具体问题的外部信息
- 当前记忆和近期对话不足以回答问题时，想补充外部知识

此技能只负责搜索并返回结果，不负责写入任何记忆。搜索结束后，应自行判断这些结果是否值得记录为活动或长期记忆。

## 支持的模式

### 1. `search`

用于搜索某个具体问题或主题。

参数：

- `action`: `"search"`
- `query`：要搜索的具体问题或主题
- `topic`：搜索主题，可选值为 `general`、`news`、`finance`
- `days`：仅在 `topic = "news"` 时用于限制最近多少天，默认 `7`
- `max_results`：返回结果数量上限，默认 `5`，最大 `10`

### 2. `extract`

用于继续展开某个具体网页，提取正文内容。

参数：

- `action`: `"extract"`
- `url`：要展开读取的网页地址
- `query`：可选，用于让 Tavily 按该意图挑选网页中更相关的内容片段
- `extract_depth`：提取深度，可选值为 `basic`、`advanced`
- `format`：返回格式，可选值为 `markdown`、`text`

## 返回结果

返回 JSON 字符串，包含：

- `search` 模式：
  - `query`：原始查询
  - `results`：结果列表，每条结果包含标题、链接、摘要内容，以及可能存在的分数和发布时间
- `extract` 模式：
  - `url`：原始网页地址
  - `results`：提取结果列表，每条结果包含链接、正文内容，以及可能存在的标题

## 注意事项

1. 一次只搜索一个明确主题，不要把多个问题混在同一次搜索里
2. 不要连续发起过多搜索，应优先围绕当前真正想知道的问题搜索
3. 只有在确认某个结果值得继续深挖时，再使用 `extract` 展开具体网页
4. 搜索和提取结果都是候选外部信息，需要自行甄别可信度和相关性，不要认为全部可信
